Odinstalowywanie oprogramowania

W systemie Windows
W systemie Mac OS X
W celu rozwiązania pewnych problemów lub po uaktualnieniu systemu operacyjnego konieczne może być odinstalowanie i ponowne zainstalowanie oprogramowania skanującego.

W systemie Windows

Uwaga:
Jeśli w systemie Windows 8.1, Windows 8, Windows 7 i Windows Vista użytkownik jest zalogowany jako użytkownik standardowy, wymagane jest konto i hasło administratora.
W systemie Windows XP należy zalogować się na konto Computer Administrator (Administrator komputera).
Wyłącz urządzenie.
Odłącz od komputera kabel interfejsu łączący urządzenie.
Wyświetl listę programów.
Windows 8.1/Windows 8:
Wybierz Desktop (Pulpit), panel Settings (Ustawienia) oraz Control Panel (Panel sterowania). Wybierz opcję Uninstall a program (Odinstaluj program) z kategorii Programs (Programy).
Windows 7/Windows Vista:
Kliknij Start i wybierz Control Panel (Panel sterowania). Kliknij opcję Uninstall a program (Odinstaluj program) z kategorii Programs (Programy).
Windows XP:
Kliknij Start i wybierz Control Panel (Panel sterowania). Dwukrotnie kliknij ikonę Add or Remove Programs (Dodaj lub usuń programy).
Wybierz z wyświetlonej listy program, który ma zostać odinstalowany.
Odinstaluj program.
Windows 8.1/Windows 8/Windows 7:
Kliknij Uninstall/Change (Odinstaluj/Zmień).
Windows Vista:
Kliknij przycisk Uninstall/Change (Odinstaluj/Zmień), a następnie kliknij polecenie Continue (Kontynuuj) w oknie User Account Control (Kontrola konta użytkownika).
Windows XP:
Kliknij Change/Remove (Zmień/Usuń).
Gdy zostanie wyświetlone okno potwierdzenia, kliknij przycisk Yes (Tak).
Postępuj zgodnie z instrukcjami wyświetlanymi na ekranie.
Może zostać wyświetlony komunikat monitu o ponowne uruchomienie komputera. W takim przypadku upewnij się, że wybrano pozycję I want to restart my computer now (Chcę teraz ponownie uruchomić komputer) i kliknij przycisk Finish (Zakończ).

W systemie Mac OS X

Uwaga:
W celu odsintalowania oprogramowania musisz pobrać Uninstaller.
Przejdź do strony:
http://www.epson.com
W tym celu należy wybrać część poświęconą pomocy technicznej w witrynie lokalnej firmy Epson.
Aby możliwe było odinstalowanie aplikacji, należy zalogować się na koncie Computer Administrator (administratora komputera).
Programów nie można odinstalowywać z poziomu konta użytkownika o ograniczonych uprawnieniach.
W zależności od aplikacji Uninstaller może być oddzielony od instalatora.
Zamknij wszystkie uruchomione aplikacje.
Kliknij dwukrotnie ikonę Uninstaller na dysku twardym Mac OS X.
Zaznacz pola wyboru oprogramowania, które chcesz odinstalować, takie jak napęd drukarki i aplikację z wyświetlanej listy.
Kliknij Uninstall.
Postępuj zgodnie z instrukcjami wyświetlanymi na ekranie.
Jeśli nie można znaleźć oprogramowania do odinstalowania w oknie Uninstaller, kliknij dwukrotnie folder Applications (Programy) na dysku twardym Mac OS X, wybierz aplikację, która ma zostać odinstalowana, a następnie przeciągnij ją do ikony Trash (Kosz).
Uwaga:
Jeśli odinstalujesz sterownik skanera, a nazwa produktu wciąż będzie wyświetlana w oknie Print & Scan (Drukuj i skanuj) (dla Mac OS X 10.8 lub 10.7) lub Print & Fax (Drukuj i faksuj) (dla Mac OS X 10.6 lub 10.5), zaznacz nazwę produktu i kliknij przycisk - usunięcia.
