Używanie skanera jako kopiarki

Ze skanera i drukarki podłączonych do komputera można korzystać tak jak z kopiarki. Zobacz dział pomocy Copy Utility odnośnie do szczegółów korzystania z oprogramowania.
Windows 8.1/Windows 8:
Wprowadź nazwę oprogramowania w panelu search (szukaj), a następnie wybierz wyświetloną ikonę.
Windows 7/Windows Vista/Windows XP:
Kliknij przycisk start, a następnie wybierz All Programs (Wszystkie programy) lub Programs (Programy) > EPSON Software > Copy Utility.
Mac OS X:
Wybierz Go (Przejdź) > Applications (Aplikacje) > Epson Software > Copy Utility.
